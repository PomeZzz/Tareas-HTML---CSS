PREGUNTAS 1

¿Cual es la diferencia entre la web e internet?
Internet es una inmensa red de computadoras alrededor de todo el mundo conectadas entre si.
cambio la web (la World wide web) es una enorme coleccion de paginas que se asientan sobre esa red de computadoras.
Asi que cundo navegas a traves de tu celular o computadora usas internet para acceder a la web.
¿Que es HTML?
Es un lenguaje de marcado que define la estructura de tu contenido.
HTML consiste en una serie de elementos que usarás para encerrar diferentes partes del contenido para que se vean o comporten de una determinada manera.
Las etiquetas de encierre pueden hacer de una palabra o una imagen un hipervínculo a otro sitio, se pueden cambiar palabras a cursiva, agrandar o achicar la letra, etc
¿Como es la estructura basica de HTML?
La estructura basica de HTML esta conformada por als estuquetas html, head y body
¿Que es una etiqueta vacia?
Una etiqueta vacía, también conocida como etiqueta autónoma o etiqueta sin contenido. Estos son algunos ejemplos: br img entre otras.

PREGUNTA 6

Aquí hay una descripción general del flujo de información:

Cliente:
envía una petición al servidor para acceder a una página web.
Servidor:
El servidor procesa la petición y envía una respuesta que contiene los recursos solicitados.
Cliente:
El cliente recibe la respuesta y procesa los archivos para mostrar la página web.
Usuario:
El usuario interactúa con la página web cargada en el cliente.

Este proceso se repita para cada nueva peticion

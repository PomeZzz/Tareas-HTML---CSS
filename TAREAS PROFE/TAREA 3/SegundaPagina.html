Ejemplos de la etiqueta

A continuación, se muestran algunos ejemplos de uso de la etiqueta SUP en HTML:

  • Para mostrar exponentes en fórmulas matemáticas, como en la fórmula para calcular el área de un círculo: A=πr2
  • Para representar los números ordinales en fórmulas matemáticas, como en 103 para indicar el valor de 10 elevado a la tercera potencia.
  • Para indicar la notación científica, como en 6.626 x 10-34 J-s para mostrar el valor de la constante de Planck.

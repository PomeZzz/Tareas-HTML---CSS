Ejemplo de la etiqueta d1

HTML
HyperText Markup Language (lenguaje de marcas de hipertexto) es el lenguaje de programación utilizado para crear páginas web.
CSS
Cascading Style Sheets (hojas de estilo en cascada) se utiliza para controlar la presentación visual de las páginas web creadas con HTML.
JAVASCRIPT
JavaScript es un lenguaje de programación utilizado para crear interactividad en las páginas web, como la validación de formularios y la creación de efectos visuales.
